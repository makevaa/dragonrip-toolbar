To change the toolbar contents, edit the toolbarItems global array. See the toolbarItems in the code to see how the keywords are used to create the toolbar.

Possible keywords for toolbar items:

  • home, bank, prof, shop, combat, market, quests, events, dungeon
  • mining, smithing, jewels, fishing, hunter, herbs, cooking, crafting, alchemy, slayer, summoning, explo, woodwork, magic, beastmastery
  • [] adds an empty square slot
  • | adds a shorter separator space
